Describing An Object

To describe an Object use these words. In the below list we are describing an object with several words

  • This wood is in round/square/oval/rectangular/....
  • Does this vehicle use petrol/diesel/gas/electric/....
  • Is it made from steel/iron/copper/plastic/silver/....?
  • It is big/small/heavy/light/short/large...?
  • This car is cheap/expensive/comfortable/big/....?
  • Do you nedd to refill/recharge/wash/clean it everyday?
  • You can get it in a fancy store/market/ electronic store/ e-commerce/...

Describing A Person

  In some situations we describing people based on their physical appearence. In English, there are some words to describe a person. In general, we describe people based on their height, hair, eyes, walking style, etc., The following lists are words to describe a person in various ways. These words can be classified into words that mark out a person's appearance like height, eyes, nose, hair, face, legs

Arm:
long arm, short arm, lank, bony, round shaped, strong, well shaped etc.,
Build:
large,small, heavy, slim, solid, strong, delicate, powerful etc.,
height:
tall, short, peak, big, lanky etc.,
Cheeks:
chubby, wide, plump, sunken, with dimples, soft, smooth, shaven, unshaven, pale etc.,
Chin:
shaven, round, decide, clean shaven, massive etc.,
Legs:
long, strong, delicate, perfect, short etc.,
Ears:
curving, earlobe, floppy,pointed
