Tree Diagram

Tree Diagram is one of the non-verbal form we used mostly. This form is simple and easy to understand. It is mostly used when the information contains lots of Branches for the same Item. Generally, the Main Item contains so many Branches and each Branch may also contain multiple branches. Putting all information in a paragraph in this format is hard for readers. Instead of putting all information in a paragraph Tree Diagram is the best method to keep information. It removes all confusion and makes it lightly to read.

  Consider an example, take the movie as the main item and movie had different genres (assume that genres are branches) like comedy, romance,drama, horror, and more and each genre has so many movies (assume that movies are sub-branches) so this type of information is messy when we write as a paragraph in this situation Tree Diagrams are the best
